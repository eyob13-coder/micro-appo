# Micro App

## Overview
A Next.js 16 application using React 19, Tailwind CSS 4, and shadcn/ui components.

## Project Structure
- `app/` - Next.js App Router pages and layouts
- `components/` - React components (including shadcn/ui)
- `lib/` - Utility functions
- `public/` - Static assets

## Tech Stack
- Next.js 16.1.4
- React 19.2.3
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui components

## Development
- Dev server runs on port 5000
- Run with `npm run dev`

## Production
- Build with `npm run build`
- Start with `npm run start` (runs on port 5000)
